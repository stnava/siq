SIQ (Super-resolution Image Quality)

A Deep Learning library for Perceptual Resampling and Super Resolution with ANTsPyX

Introduction

Welcome to SIQ! This package utilizes Deep Back-Projection Networks (DBPN) and VGG-style perceptual loss networks to construct state-of-the-art super-resolution images, specifically tailored for medical imaging and ANTs workflows.

1. The Magical siq.auto Mode

For new users, the easiest way to start is using the new siq.auto functionality. It intelligently figures out your image dimensionality and upsamples it with an optimal baseline model.

How to use auto mode:

import siq

# Just provide the path or an ANTs image!
high_res_image = siq.auto('path/to/my_image.nii.gz', target_resolution=(256, 256, 256))

What it does: It loads your image, determines if it's 2D or 3D, constructs a default Deep Back-Projection Network (DBPN), runs the upsampling inference, and optionally resamples exactly to your target resolution.

2. Core Features

Deep Back-Projection Network (DBPN)

The DBPN is the core architecture in SIQ. It iteratively projects features up and down to refine the high-resolution estimate. You can create customized 2D and 3D DBPN architectures using siq.dbpn() and siq.default_dbpn().

VGG & ResNet Perceptual Loss

Instead of relying solely on Mean Squared Error (MSE), SIQ integrates pre-trained ResNet and VGG networks (pseudo-3D transferred weights) to evaluate image quality based on deep feature maps. Access these with siq.pseudo_3d_vgg_features() and siq.get_grader_feature_network().

Automated Training Loops

SIQ handles the complexity of memory management and patch extraction with fully automated generators: siq.train() and siq.train_seg() for simultaneous super-resolution and segmentation tasks.

3. Advanced: Training a Custom Model

For research, you often need to train a model on your specific domain data. Here's a quick start:

import siq

# Create a 3D DBPN model (e.g. 2x upsampling)
model = siq.default_dbpn(strider=[2, 2, 2], dimensionality=3, option='small')

# Train using patches
siq.train(
    mdl=model,
    filenames_train=['train_1.nii.gz', 'train_2.nii.gz'],
    filenames_test=['test_1.nii.gz'],
    target_patch_size=[32, 32, 32],
    target_patch_size_low=[16, 16, 16],
    output_prefix='my_super_model'
)

4. Evaluation & Utilities

SIQ also provides utilities to evaluate your models against others:

  • siq.compare_models(): Compare different `.keras` or `.h5` models on the same image.
  • siq.region_wise_super_resolution(): Super-resolve massive images by breaking them into overlapping tiles and stitching them back seamlessly.
  • siq.simulate_image(): Downsample and add noise to high-resolution images to create training data pairs.
