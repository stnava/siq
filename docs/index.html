The siq.auto Pipeline

Medical image processing shouldn't require a Ph.D. in computer science. The `siq.auto` interface acts as an intelligent entry point, automatically parsing your data format, inferring the correct upsampling parameters, and routing the task to the optimal pre-trained DBPN model.

With one function call, you can achieve visual fidelity previously only available through extended scanner acquisition times.

import ants import siq # 1. Load a low-resolution scan img = ants.image_read('patient_t1_lowres.nii.gz') # 2. Let SIQ figure out the rest # auto() handles dimensionality, scaling, and inference img_sr = siq.auto( image=img, modality='T1', verbose=True ) # 3. Save the structurally enhanced output ants.image_write(img_sr, 'patient_t1_super_res.nii.gz')
from siq import region_wise_super_resolution # Perform targeted upsampling on specific labels # Useful for high-detail hippocampus or tumor studies enhanced_roi = region_wise_super_resolution( image=img, mask=brain_mask, super_res_model=my_dbpn_model, dilation_amount=4 )

Advanced Segmentation-Guided SR

Sometimes you don't need the whole brain upsampled. SIQ includes advanced modules like `region_wise_super_resolution` which restricts processing to regions of interest, blending the high-resolution output seamlessly into the original background space.

This approach drastically reduces GPU memory overhead for massive 3D volumes.
